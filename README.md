# vip-caijing-journal

本仓库用于归档和发布多个独立 HTML 页面。页面内容来自公开的财经信息整理，不针对任何单一域名或特定网站。仓库主要用作静态存档和展示用途。

## 项目简介

`vip-caijing-journal` 是一个以 HTML 页面为单位的归档仓库。页面内容以财经类信息为主，涵盖日常整理的各类公开资料。每个页面均为独立完整的 HTML 文件，可直接通过浏览器打开查看，无需额外依赖。

## 目录结构说明

```
.
├── index.html          # 仓库主页 / 页面索引
├── pages/              # 归档的 HTML 页面存放目录
│   ├── page-001.html
│   ├── page-002.html
│   └── ...
├── assets/             # 资源文件（图片、样式、脚本等）
│   ├── css/
│   ├── js/
│   └── images/
└── README.md           # 本文件
```

- `pages/`：存放所有归档的 HTML 页面文件，按编号或日期命名。
- `assets/`：存放页面引用的外部资源，如样式表、脚本和图片。
- `index.html`：提供页面导航和索引，方便浏览所有归档内容。

## 页面归档说明

- 页面可能包含时效性信息，不保证实时更新或长期有效。

## 使用方式

1. 克隆仓库到本地：
   ```bash
   git clone https://github.com/your-username/vip-caijing-journal.git
   ```
